How does JavaScript compare to HTML and CSS?

Javascript is completely differnent syntax of language to HTML and CSS, but both work together. HTML is the markup language, CSS is the style language and Javascript is the programming language.

With a webpage, Javascript is what programs the HTML elements based on the users interactions.

Control flow and loops

Control flow is used with If "true" or "false" statements, they are a way to execute code based on a condition, these can be multiple loops at a time to generate a solution. For example, using a daily routine of deciding what to wear based on weather conditions.

  • If temperature < 20 + rain - wear warm clothes, take umbrella
  • If else temperature > 20 + no rain, wear normal clothes, no umbrella

    • Another example of a control flow for meal times.

  • if time < 10 - make breakfast.
  • if else time > 12 - make lunch.
  • if else time > 19 - make dinner.

If either statements are true this is your course of action.

Arrays and objects

Arrays have indexes which are an integer value, whereas objects have keys which are generally a string. You can access arrays by using varName[3] for example, whereas with an object it's varName.keyName or also varName["keyName"].

Objects take up more space and are harder to garbage collect and work like a hashtable so finding what you need is potentially easier cause you don't need to iterate to find it.
