HTML CSS DOM Pt.1

Can you think of an analogy to describe HTML, CSS and the DOM? How would you describe it to your non-technical friend?

HTML stands for Hyper Text Markup Language - it is the language of certain codes and symbols used to structure the content on web pages.

CSS stands for Cascading Style Sheets - it works with the HTML for styling the webpage, colours, fonts, borders and layouts.

DOM stands for Document Object Model - it is the programming behind HTML and CSS that makes everything work together on the webpage.

What is meant by boxifying design?

Every part of the webpage depending on the content is seperated into boxes using CSS to arrange the HTML. A webpage is like a sheet of paper, each part of the page can only be cut up into rectangles to keep all content in order.

What is the box model?

The box model is like the hierarchy of the content in which each box is placed. The top is the title within its own rectangle. The body is below the title that has all of the information, each paragraph evenly placed within its own box. The border is the white space around the page, the margin is the size between the border and content while the padding is the size of the space around all boxes.
